Time transforms reshape when events happen. `palindrome` reverses the pattern every other cycle, creating a natural back-and-forth motion.

## Code

<CodeComparison>

```ts
note("c3 e3 g3 b3")

  .release(0.15)
```

## What's Happening

- `note("c3 e3 g3 b3")` uses note names instead of numbers — C3, E3, G3, B3 (a Cmaj7 arpeggio)
- `.s("saw")` selects a sawtooth wave — brighter and buzzier than sine
- `.fast(2)` doubles the pattern speed: 8 notes per cycle
- `.palindrome()` reverses the pattern every other cycle (up-down-up-down)
- `.lpf(800)` applies a low-pass filter at 800Hz, taming the sawtooth brightness
- The combination creates a smooth arpeggio that ascends and descends continuously
