The simplest Tussel pattern: a single note played on a sine oscillator. This is the "hello world" of livecoding.

## What's Happening

- `note("0")` creates a pattern that plays MIDI note 0 (C) every cycle
- `.s("sine")` selects the sine wave synthesizer
- `.release(0.3)` sets a 300ms release envelope so the sound fades out smoothly
- `setcps(0.5)` sets the tempo to 0.5 cycles per second (one cycle = 2 seconds)

This is the foundation everything else builds on: a **pattern** that produces **events** at regular intervals, routed to a **sound source**.
