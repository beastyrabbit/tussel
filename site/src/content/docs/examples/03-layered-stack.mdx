`stack` plays multiple patterns simultaneously — like tracks in a DAW running in parallel.

## Code

<CodeComparison>
  <Fragment slot="tussel">
```ts
setcps(0.5);
stack(
  s("bd hh sd hh"),
  note("0 2 4 7").s("triangle")
    .gain(0.3).release(0.2)
)
```
  </Fragment>
  <Fragment slot="strudel">
```js

stack(
  s("bd hh sd hh"),

)
```
  </Fragment>
  <Fragment slot="tidal">
```haskell
setcps 0.5
d1 $ sound "bd hh sd hh"

  # gain 0.3

```

## What's Happening

- `stack(...)` combines two patterns so they play at the same time
- The first pattern uses `s()` (sound) with sample names: kick, hi-hat, snare, hi-hat
- The second pattern uses `note()` with a triangle wave synth
- `.gain(0.3)` reduces the synth volume so the drums stay prominent
- In Tidal, stacking is done by assigning patterns to separate channels (`d1`, `d2`)
