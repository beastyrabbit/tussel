## What's Happening

- `"0 2 4 7"` is **mini notation** — four values separated by spaces are evenly distributed across one cycle
- Each number is a MIDI note offset: C, D, E, G (a major pentatonic fragment)
- The pattern repeats every cycle, creating a continuous melodic loop
- `.release(0.2)` gives each note a short fade-out so they don't clip
